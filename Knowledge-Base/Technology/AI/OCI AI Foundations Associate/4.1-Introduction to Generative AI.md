# 4.1-Introduction to Generative AI

So what is AI? AI is the ability of machine learnings to imitate human intelligence. Machine learning is a subset of AI where algorithms are used to learn from past data, and predict outcome on new data, or to identify trends from the past data.

Deep learning is a further subset of machine learning in which algorithms are modeled to learn from complex data using neural networks. So we have looked at AI, machine learning, and deep learning in the previous modules. And in this module, we will look at generative AI in more details.

So as the name specifies, generative AI is a type of AI that can create new content. Generative AI includes models that can create a wide range of outputs such as text and any kind of media, whether it is images, music, videos, and other types of data. GenAI, as you can see from the graphic here, is a subset of deep learning where the models are trained to generate output on their own. And GenAI unlocks exciting possibilities for creative tasks, automation, and new ideas.

So how does generative AI really work? Generative AI at a very high level, a generative AI model learns the underlying patterns in a given data set and uses that knowledge to create new data that shares those patterns. To explain this in a simple way, let's imagine you're trying to teach a generative AI model how to draw a dog.

You would start by giving it lots of pictures of dogs to learn from. The GenAI model doesn't know anything about what a dog looks like. But by looking at these pictures, it starts to figure out common patterns and features, like dogs have pointy ears, sharp teeth, whiskers, et cetera.

Once the GenAI model has trained on enough dog pictures, you can then ask it to draw a new picture of a dog. And the GenAI model will use the patterns it has learned to generate a picture that hopefully looks like a dog. But remember, the GenAI model is not copying any of the pictures it has seen before. But it is creating a new image based on the patterns it has learned.

In practice, this process involves a lot of complex math and computation. And there are different techniques and architectures that can be used. But this is the basic idea behind generative AI. So how does it compare to machine learning? As we have learned in the previous modules, machine learning is about identifying patterns-- in this case, pictures-- and using those patterns to recognize and classify new examples of cats and dogs.

Like in the previous example, machine learning starts with the model being trained with the training data set. So here you see training where the model is trying to learn new capability from existing data. But now the training data set consists of a set of input features and output labels, so what we refer to as data and labels on this slide.

These features could be body shape, skin texture, eye color, et cetera. So in this case, like in the previous example, we provide pictures of a dog or a cat, but we also provide a corresponding output label. So here, there is a picture of a cat, and the label says cat. There is a dog, and the label says dog, and so on, and so forth.

During the process of training, machine learning model learns the relationship between input features and the corresponding output labels from the provided data. Once the model learns from this data, we have what we refer to as a trained model. After the model is trained, it can be used for making predictions, or what we refer to as inference. Inference is a process of getting a prediction by giving a new data point.

So in this example, we provide a picture of a cat or a dog. And the trained model predicts the output is a dog. So see here, there is no label. We just provide a picture. And then we ask, what is this picture about? And then the trained model predicts that the output, it gives a label, saying it's a picture of a dog.

Now, this is a type of machine learning which is referred to as supervised machine learning. So hopefully, based on those last two slides, you can see the difference between a generative AI model and a traditional machine learning model.

So in the case of a traditional machine learning model, the model learns relationship between data and label. And the output is a label, a prediction, or a classification. In the case of a GenAI model, the model is learning patterns in unstructured content. Unlike traditional machine learning, GenAI models don't require labeled data in its pre-training stage. It is pre-trained on unlabeled data.

Now, in later lessons, this is a more advanced topic. You will learn that a pre-trained GenAI model can be further trained on a labeled data set to obtain what we refer to as a fine-tuned model for specific tasks. But for the purposes of introducing you to generative AI, you can assume that the GenAI models are learning patterns based on unstructured content.

And the use cases are different. A traditional machine learning model can be used for classification. It could be used for making predictions, et cetera, while a GenAI model is used for generation. So it could be text generation, or it could be any kind of media generation.

Now, what are the types of generative AI models? There are two main types of GenAI models. The first type is text-based, where models can generate text, they can generate code, they can generate dialogues or chat, or they can generate articles. In this case, the models are learning from large collections of text data to capture patterns, language structures, and semantic relationships.

The second kind of GenAI models are what we refer to as multimodal models. These are the models which are capable of processing and generating multiple modalities and learning from multiple modalities simultaneously. These modalities could include text, images, audio, and video. So these are the two types of generative models based on a very high level understanding.

And finally, what are the applications for generative AI? Well, there is every vertical industry and every horizontal use case is being touched by generative AI. We are already seeing its use in generating, creating content, creative content. We are already seeing GenAI being used for image and video generation. And there's literally every use case you can think of.

One use case where GenAI is really helping a lot is medical imaging and drug discovery. GenAI is greatly accelerating the rate of scientific progress in medicine, offering breakthrough in drug development, diagnosis, and increasing, overall, the rate of scientific progress.

And that's pretty much it. Hopefully, in this lesson, you got a good overview of what generative AI is and how generative AI differs from a traditional AI approach. I hope you found this lesson useful. Thanks for your time.

# 4.1-Introducción a la IA generativa

¿Qué es la IA? La IA es la capacidad del aprendizaje automático para imitar la inteligencia humana. El aprendizaje automático es un subconjunto de la IA donde se utilizan algoritmos para aprender de datos históricos y predecir resultados con datos nuevos, o para identificar tendencias a partir de datos anteriores.

El aprendizaje profundo es una rama del aprendizaje automático en la que los algoritmos se diseñan para aprender de datos complejos mediante redes neuronales. En los módulos anteriores, hemos analizado la IA, el aprendizaje automático y el aprendizaje profundo. En este módulo, profundizaremos en la IA generativa.

Como su nombre indica, la IA generativa es un tipo de IA capaz de crear contenido nuevo. Incluye modelos que pueden generar una amplia gama de resultados, como texto y cualquier tipo de contenido multimedia, ya sean imágenes, música, vídeos u otros datos. La IA generativa, como se puede apreciar en el gráfico, es un subconjunto del aprendizaje profundo, donde los modelos se entrenan para generar resultados de forma autónoma. La IA generativa abre un abanico de posibilidades fascinantes para tareas creativas, automatización y nuevas ideas.

¿Cómo funciona realmente la IA generativa? A grandes rasgos, un modelo de IA generativa aprende los patrones subyacentes en un conjunto de datos determinado y utiliza ese conocimiento para crear nuevos datos que comparten esos patrones. Para explicarlo de forma sencilla, imaginemos que intentamos enseñarle a un modelo de IA generativa a dibujar un perro.

Para empezar, le proporcionarías muchas imágenes de perros para que aprendiera. El modelo GenAI no sabe nada sobre la apariencia de un perro. Pero al observar estas imágenes, comienza a identificar patrones y características comunes, como orejas puntiagudas, dientes afilados, bigotes, etcétera.

Una vez que el modelo GenAI se haya entrenado con suficientes imágenes de perros, puedes pedirle que dibuje una nueva imagen. El modelo GenAI utilizará los patrones aprendidos para generar una imagen que, idealmente, se parezca a un perro. Pero recuerda, el modelo GenAI no copia ninguna de las imágenes que ha visto antes, sino que crea una nueva imagen basada en los patrones aprendidos.

En la práctica, este proceso implica cálculos matemáticos complejos y diversas técnicas y arquitecturas. Esta es la idea básica de la IA generativa. ¿En qué se diferencia del aprendizaje automático? Como hemos visto en los módulos anteriores, el aprendizaje automático consiste en identificar patrones (en este caso, imágenes) y utilizarlos para reconocer y clasificar nuevos ejemplos de gatos y perros.

Al igual que en el ejemplo anterior, el aprendizaje automático comienza con el entrenamiento del modelo utilizando el conjunto de datos de entrenamiento. Aquí vemos el entrenamiento, donde el modelo intenta aprender nuevas capacidades a partir de los datos existentes. En este caso, el conjunto de datos de entrenamiento consta de un conjunto de características de entrada y etiquetas de salida, a lo que nos referimos como datos y etiquetas en esta diapositiva.

Estas características podrían ser la forma del cuerpo, la textura de la piel, el color de los ojos, etcétera. En este caso, como en el ejemplo anterior, proporcionamos imágenes de un perro o un gato, pero también una etiqueta de salida correspondiente. Así, aquí hay una imagen de un gato y la etiqueta dice "gato". Hay una imagen de un perro y la etiqueta dice "perro", y así sucesivamente.

Durante el proceso de entrenamiento, el modelo de aprendizaje automático aprende la relación entre las características de entrada y las etiquetas de salida correspondientes a partir de los datos proporcionados. Una vez que el modelo aprende de estos datos, obtenemos lo que denominamos un modelo entrenado. Tras el entrenamiento, el modelo puede utilizarse para realizar predicciones, lo que se conoce como inferencia. La inferencia es el proceso de obtener una predicción a partir de un nuevo dato.

En este ejemplo, proporcionamos una imagen de un gato o un perro. El modelo entrenado predice que la imagen es un perro. Como pueden ver, no hay etiqueta. Simplemente proporcionamos una imagen. Luego preguntamos: ¿De qué trata esta imagen? El modelo entrenado predice que la imagen es un perro.

Ahora bien, este es un tipo de aprendizaje automático conocido como aprendizaje automático supervisado. Así que, con base en las dos últimas diapositivas, espero que puedan apreciar la diferencia entre un modelo de IA generativa y un modelo de aprendizaje automático tradicional.

En el caso de un modelo de aprendizaje automático tradicional, el modelo aprende la relación entre los datos y las etiquetas. El resultado es una etiqueta, una predicción o una clasificación. En el caso de un modelo GenAI, el modelo aprende patrones en contenido no estructurado. A diferencia del aprendizaje automático tradicional, los modelos GenAI no requieren datos etiquetados en su etapa de preentrenamiento. Se preentrenan con datos sin etiquetar.

En lecciones posteriores, este tema será más avanzado. Aprenderás que un modelo GenAI preentrenado puede entrenarse aún más con un conjunto de datos etiquetados para obtener lo que llamamos un modelo optimizado para tareas específicas. Sin embargo, para introducirte en la IA generativa, puedes asumir que los modelos GenAI aprenden patrones a partir de contenido no estructurado.

Los casos de uso son diferentes. Un modelo de aprendizaje automático tradicional se puede usar para la clasificación, para hacer predicciones, etc., mientras que un modelo GenAI se usa para la generación. Por ejemplo, puede generar texto o cualquier tipo de contenido multimedia.

Ahora bien, ¿qué tipos de modelos de IA generativa existen? Hay dos tipos principales de modelos GenAI. El primero se basa en texto, donde los modelos pueden generar texto, código, diálogos o chats, o artículos. En este caso, los modelos aprenden de grandes conjuntos de datos textuales para capturar patrones, estructuras lingüísticas y relaciones semánticas.

El segundo tipo de modelos GenAI son los que denominamos modelos multimodales. Estos modelos son capaces de procesar y generar múltiples modalidades, y de aprender de ellas simultáneamente. Estas modalidades pueden incluir texto, imágenes, audio y vídeo. Por lo tanto, estos son los dos tipos de modelos generativos basados en una comprensión de muy alto nivel.

Y, por último, ¿cuáles son las aplicaciones de la IA generativa? Pues bien, la IA generativa está presente en todos los sectores y en todos los ámbitos. Ya la vemos aplicada a la generación y creación de contenido creativo . También la utilizamos para la generación de imágenes y vídeos. Y, literalmente, existen todas las aplicaciones imaginables.

Un caso de uso en el que la IA genómica está siendo de gran ayuda es en la obtención de imágenes médicas y el descubrimiento de fármacos. La IA genómica está acelerando enormemente el ritmo del progreso científico en medicina, ofreciendo avances significativos en el desarrollo de fármacos y el diagnóstico, e incrementando, en general, el ritmo del progreso científico.

Y eso es todo. Espero que en esta lección hayas obtenido una buena visión general de qué es la IA generativa y en qué se diferencia de la IA tradicional. Espero que te haya resultado útil. Gracias por tu tiempo.

# Introdução a IA Generativa

O que é IA? IA é a capacidade do aprendizado de máquina de imitar a inteligência humana. O aprendizado de máquina é um subconjunto da IA, onde algoritmos são usados para aprender com dados passados e prever resultados com base em novos dados, ou para identificar tendências a partir de dados passados.

Aprendizado profundo é um subconjunto do aprendizado de máquina no qual os algoritmos são modelados para aprender com dados complexos usando redes neurais. Nos módulos anteriores, analisamos IA, aprendizado de máquina e aprendizado profundo. Neste módulo, vamos explorar a IA generativa com mais detalhes.

Como o próprio nome indica, a IA generativa é um tipo de IA capaz de criar conteúdo novo. A IA generativa inclui modelos que podem gerar uma ampla gama de resultados, como texto e qualquer tipo de mídia, sejam imagens, músicas, vídeos e outros tipos de dados. A IA generativa, como você pode ver no gráfico, é um subconjunto do aprendizado profundo, no qual os modelos são treinados para gerar resultados por conta própria. E a IA generativa abre possibilidades incríveis para tarefas criativas, automação e novas ideias.

Então, como a IA generativa realmente funciona? Em termos gerais, um modelo de IA generativa aprende os padrões subjacentes em um determinado conjunto de dados e usa esse conhecimento para criar novos dados que compartilham esses padrões. Para explicar isso de forma simples, imagine que você está tentando ensinar um modelo de IA generativa a desenhar um cachorro.

Você começaria fornecendo ao modelo muitas fotos de cachorros para que ele aprendesse. O modelo GenAI não sabe nada sobre a aparência de um cachorro. Mas, ao observar essas fotos, ele começa a identificar padrões e características comuns, como orelhas pontudas, dentes afiados, bigodes, etc.

Depois que o modelo GenAI tiver sido treinado com um número suficiente de imagens de cachorros, você poderá pedir que ele desenhe um novo cachorro. O modelo GenAI usará os padrões que aprendeu para gerar uma imagem que, idealmente, se pareça com um cachorro. Mas lembre-se: o modelo GenAI não está copiando nenhuma das imagens que viu antes. Ele está criando uma nova imagem com base nos padrões que aprendeu.

Na prática, esse processo envolve muita matemática e computação complexas. E existem diferentes técnicas e arquiteturas que podem ser usadas. Mas essa é a ideia básica por trás da IA generativa. Então, como ela se compara ao aprendizado de máquina? Como aprendemos nos módulos anteriores, o aprendizado de máquina consiste em identificar padrões — neste caso, imagens — e usar esses padrões para reconhecer e classificar novos exemplos de gatos e cachorros.

Assim como no exemplo anterior, o aprendizado de máquina começa com o modelo sendo treinado com o conjunto de dados de treinamento. Aqui, vemos o treinamento, onde o modelo tenta aprender novas capacidades a partir de dados existentes. Mas, neste caso, o conjunto de dados de treinamento consiste em um conjunto de características de entrada e rótulos de saída, que chamamos de dados e rótulos neste slide.

Essas características podem ser formato do corpo, textura da pele, cor dos olhos, etc. Então, neste caso, como no exemplo anterior, fornecemos fotos de um cachorro ou de um gato, mas também fornecemos um rótulo de saída correspondente. Aqui, temos a foto de um gato, e o rótulo diz "gato". Temos um cachorro, e o rótulo diz "cachorro", e assim por diante.

Durante o processo de treinamento, o modelo de aprendizado de máquina aprende a relação entre as características de entrada e os rótulos de saída correspondentes a partir dos dados fornecidos. Uma vez que o modelo aprende com esses dados, temos o que chamamos de modelo treinado. Após o treinamento, o modelo pode ser usado para fazer previsões, ou o que chamamos de inferência. Inferência é o processo de obter uma previsão a partir de um novo ponto de dados.

Neste exemplo, fornecemos a imagem de um gato ou de um cachorro. O modelo treinado prevê que a resposta será um cachorro. Veja, não há rótulo. Apenas fornecemos a imagem. Em seguida, perguntamos: "Sobre o que é esta imagem?". O modelo treinado prevê que a resposta será um cachorro e atribui um rótulo.

Agora, este é um tipo de aprendizado de máquina conhecido como aprendizado de máquina supervisionado. Portanto, espero que, com base nesses dois últimos slides, você consiga perceber a diferença entre um modelo de IA generativa e um modelo de aprendizado de máquina tradicional.

Assim, no caso de um modelo de aprendizado de máquina tradicional, o modelo aprende a relação entre dados e rótulos. E a saída é um rótulo, uma previsão ou uma classificação. No caso de um modelo GenAI, o modelo aprende padrões em conteúdo não estruturado. Ao contrário do aprendizado de máquina tradicional, os modelos GenAI não exigem dados rotulados em sua fase de pré-treinamento. Eles são pré-treinados com dados não rotulados.

Agora, em lições posteriores, este é um tópico mais avançado. Você aprenderá que um modelo GenAI pré-treinado pode ser ainda mais treinado em um conjunto de dados rotulados para obter o que chamamos de modelo ajustado para tarefas específicas. Mas, para fins de introdução à IA generativa, você pode assumir que os modelos GenAI aprendem padrões com base em conteúdo não estruturado.

E os casos de uso são diferentes. Um modelo tradicional de aprendizado de máquina pode ser usado para classificação. Ele pode ser usado para fazer previsões, etc., enquanto um modelo GenAI é usado para geração. Portanto, pode ser geração de texto ou qualquer tipo de geração de mídia.

Quais são os tipos de modelos de IA generativa? Existem dois tipos principais de modelos de IA generativa. O primeiro tipo é baseado em texto, onde os modelos podem gerar texto, código, diálogos ou bate-papo, ou artigos. Nesse caso, os modelos aprendem com grandes conjuntos de dados textuais para capturar padrões, estruturas de linguagem e relações semânticas.

O segundo tipo de modelos GenAI são os que chamamos de modelos multimodais. Esses modelos são capazes de processar e gerar múltiplas modalidades e aprender com elas simultaneamente. Essas modalidades podem incluir texto, imagens, áudio e vídeo. Portanto, esses são os dois tipos de modelos generativos baseados em um nível de compreensão muito elevado.

E, finalmente, quais são as aplicações da IA generativa? Bem, todos os setores verticais e todos os casos de uso horizontais estão sendo impactados pela IA generativa. Já vemos seu uso na geração e criação de conteúdo, conteúdo criativo . Já vemos a IA generativa sendo usada para geração de imagens e vídeos. E existem literalmente todos os casos de uso que você possa imaginar.

Uma das áreas em que a GenAI está sendo de grande ajuda é a de imagens médicas e descoberta de medicamentos. A GenAI está acelerando significativamente o ritmo do progresso científico na medicina, proporcionando avanços no desenvolvimento de medicamentos, diagnósticos e, de forma geral, impulsionando o progresso científico.

E é basicamente isso. Espero que, nesta lição, você tenha obtido uma boa visão geral do que é IA generativa e como ela difere de uma abordagem de IA tradicional. Espero que tenha achado esta lição útil. Obrigado pelo seu tempo.
