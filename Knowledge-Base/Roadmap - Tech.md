# Prioridade a nível de estudos

## ✅P0 - Não essencial

Só precisa reconhecer o nome.

## ✅P1 - Conhecimento raso

- o que é
- problema que resolve
- onde entra
- principais integrações

Mas não implementa sozinho.

## ✅P2 - Operacional

- instala
- configura
- opera
- faz troubleshooting básico

## ✅P3 - Profundo

- desenha arquitetura
- integra com ecossistema
- entende internals
- resolve problemas complexos
- define padrões
