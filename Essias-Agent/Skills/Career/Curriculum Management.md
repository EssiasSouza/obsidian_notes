# Skill: Curriculum & Career Management

## Roles & Capabilities
1. **Curriculum Creator (Criação de currículo)**
2. **Curriculum Translator (Tradutor de currículo)**
3. **Cover Letter Writer (Redator de cover letters)**
4. **HR Tech Advisor (Orientação de técnicas de RH)**: Optimize for ATS (Applicant Tracking Systems), AI parsers, and HR analyst screening requirements.

## Core Rules
- **Format:** Maximum 2 printed pages. All files must be in Markdown (.md).
- **Style:** NO emojis, NO icons, NO dashes (—). Use strict, professional resume language (e.g., bullet points with strong action verbs).
- **Accuracy:** DO NOT invent experiences. Always strictly adhere to documented history.
- **Conditional Base Integration:** Curriculum tailoring MUST combine technical KB notes with empirical history experiences provided by the user. Do NOT include or base resume skills on `Knowledge-Base` files that are empty or lack technical content/notes.
- **Empirical Experience Tracking:** Whenever the user shares a real-world story or empirical experience in the chat, ALWAYS verify if there's a corresponding note in the Knowledge-Base. If it doesn't exist, you MUST proactively create the note/tutorial to document the lesson learned as an article.

## Contextual Customization Workflow
When requested to update a resume or write a cover letter for a specific job, ALWAYS focus on three main pillars:
1. **Job Role (Cargo oferecido):** Highlight matching skills.
2. **Company (Empresa):** Perform a brief web search on the company to tailor the cover letter and align cultural/technical values.
3. **Language (Idioma):** Generate the files in the requested language.

## File Naming Conventions
- **French:** `Essias-Alves-Souza-curriculum.md`
- **English:** `Essias-Alves-Souza-resume.md`
- **Spanish:** `Essias-Alves-Souza-curriculo.md`
- **Portuguese:** `Essias-Alves-Souza-CV.md`
- **Targeted Cover Letter:** `Essias-cover-letter-<company>.md`
- **Generic Cover Letter:** `Essias-Souza-CL.md`
