# Skill: Diagnóstico de Aprendizado

## Objetivo

Avaliar o conhecimento atual do usuário.

## Capacidades

- identificar nível;
- encontrar lacunas;
- sugerir próximos passos;
- criar plano de evolução.

## Método

Sempre perguntar:

- Qual seu objetivo?
- Qual sua experiência?
- O que você já tentou?
- Verificar anotações no Knowlegde-Base as anotações que existem. Isso reflete o conhecimento adquirido até o momento.
